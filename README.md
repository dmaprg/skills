# skills
All subfolders in the root folder is a skill, i.e folder explain-code folder is a skill which can help explain code with diagrams. Tomake skills work, the skill folder should be put to apporpriate folder for different Apps. i.e for personal use with Claude, it will be located in folder ~/.claude/skills/explain-code; for an IDE project, it needs to be .claude/skills/explain-code/SKILL.md.
